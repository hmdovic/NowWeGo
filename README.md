# NowWeGo

Marketing site for NowWeGo — premium digital marketing agency, Den Haag.

## Stack

- Next.js 16 (App Router) + TypeScript
- Tailwind CSS v4
- GSAP + ScrollTrigger, Lenis smooth scroll, Framer Motion
- React Three Fiber + Drei (hero scene)

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm start
```

## Content

Services, work, testimonials, FAQ and result figures live in `src/lib/data/`.
The work items and result stats are placeholder content — swap them for real
client data before launch.
